# Student-Management-System-Oops
A menu-driven Student Management System using CPP and OOPS.

## Features

- Add Student
- Display Student Records
- Search Student by Roll Number
- Update Student Details
- Delete Student Records
- File Handling using `fstream`

## Concepts Used

- Classes & Objects
- Constructors
- Encapsulation
- Getters & Setters
- File Handling
- Switch Case
- Loops
- Conditional Statements

## Technologies

- C++
- VS Code
- Git & GitHub
